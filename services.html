Our Services

At TransitionPoint, our focus in on using data and web technologies to create applications that will help your business excel. We offer core services in technology strategy and analysis, information and systems architecture, software development and prototyping, and R&D into emerging technologies. We particularly like to help companies transition existing applications to more robust and scalable products.

Technology Strategy & Analysis

A fundamental requirement of many new businesses is a strong understanding of the technology that can make their processes faster, cheaper, and more efficient. We want to make sure you have the information you need to succeed, such as

  • data systems analysis and design,
  • interim CTO services, and
  • web and network systems design and implementation.

Growing companies may also need additional technology expertise to solve existing problems or improve current practices. You can use our team to provide

  • performance analysis and tuning,
  • database analysis, rearchitecture, and conversion, and
  • technical due diligence for clients and VCs

Information and Systems Architecture

Building robust, scalable business systems requires strong architecture skills at both the information and systems level. We can provide

  • requirements analysis
  • data architecture and redesign
  • network systems architecture and redesign
  • software systems analysis and redesign

We are often asked to port "home-grown" web database applications to more sophisticated application platforms and system architectures for early-stage technology companies.

Software Development and Prototyping

Software development is the common denominator for almost all of our interactions with clients. We strive to be platform and vendor neutral in our development efforts. Our services range from the planning and implementation phases to performance testing on the following key technologies:

  • ColdFusion
  • Active Server Pages
  • Java Server Pages
  • Java Servlets
  • PHP
  • Perl

We also are geared for extremely rapid prototyping and development efforts. This service, combined with willingness for equity participation in client companies, is extremely attractive to many startups at the friends-and-family or angel rounds. We can in many cases have functional prototypes in one week or less.

R&D for Emerging Technologies

We strongly invest in new technologies that are oriented towards data and web technologies. We can offer advice, comparative analysis, and development skills for new technologies including

  • WAP and other wireless technologies
  • embedded Java applications
  • new web database technology platforms
